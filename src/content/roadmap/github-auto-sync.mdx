title: GitHub Auto-Sync Stats
description: Automatically fetch and cache repository metadata (stars, forks, contributors, last commit) for public NukeHub projects.
status: shipped
targetQuarter: Q2 2025
relatedProject: nukehub.org
order: 2
---

A scheduled GitHub Actions workflow refreshes project stats every six hours and writes them to a static JSON cache that the Astro build reads at generation time.
